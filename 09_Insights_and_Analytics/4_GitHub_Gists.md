# GitHub Gists

GitHub Gists are used to share small pieces of code or text.

They are useful for quickly sharing code snippets, configuration files, and notes.

---

## What Gists Are Used For

Developers commonly use GitHub Gists for:

- sharing code snippets
- saving useful scripts
- sharing configuration files
- posting examples in discussions or tutorials

---

## Types of Gists

GitHub supports two types of Gists.

### Public Gists

Public Gists are visible to everyone.

Anyone with the link can view and use the code.

### Secret Gists

Secret Gists are not listed publicly but can be accessed by anyone with the link.

---

## Example Use Case

A developer writes a small script and shares it as a Gist.

Other developers can view, copy, and use the script.

---

## Benefits

GitHub Gists help developers:

- share code quickly
- store small reusable scripts
- collaborate on simple code snippets
