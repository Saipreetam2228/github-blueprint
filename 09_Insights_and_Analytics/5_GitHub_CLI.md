# GitHub CLI

GitHub CLI is a command line interface that allows developers to interact with GitHub directly from the terminal.

Instead of using the web interface, developers can manage repositories, issues, and pull requests using simple commands.

---

## Why GitHub CLI is Useful

GitHub CLI helps developers:

- manage repositories faster
- create issues from the terminal
- manage pull requests
- automate workflows

---

## Installing GitHub CLI

GitHub CLI can be installed from the official website or package managers.

After installation, developers can authenticate their GitHub account.

---

## Example Commands

Some common GitHub CLI commands include:

```
gh repo create
gh issue list
gh pr create
gh pr merge
```

These commands allow developers to interact with GitHub directly from the terminal.

---

## Example Workflow

A developer can:

1. create a repository
2. open an issue
3. create a pull request
4. merge the pull request

All using the terminal.

---

## Benefits

GitHub CLI provides:

- faster repository management
- automation through scripts
- improved developer productivity
