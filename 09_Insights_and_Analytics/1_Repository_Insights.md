# GitHub Repository Insights

GitHub provides analytics tools that help developers understand how their repository is being used.

These analytics are available in the **Insights** section of a repository.

---

## What Insights Show

GitHub Insights provide information about:

- repository activity
- contributions
- project growth
- user engagement

---

## Types of Insights

GitHub provides different types of analytics.

### Contributors

Shows the list of developers who contributed to the project.

It displays:

- commits per contributor
- contribution history
- activity graph

---

### Commit Activity

Shows how frequently commits are made to the repository.

It includes:

- weekly commit graphs
- commit trends over time

---

### Traffic

Traffic insights show repository visitors.

Information includes:

- number of visitors
- repository views
- clone statistics

---

### Forks and Stars

Insights also show:

- how many people forked the repository
- how many users starred the project

These metrics help measure project popularity.

---

## Benefits of Insights

GitHub Insights help developers:

- track project growth
- analyze contribution patterns
- understand community engagement
