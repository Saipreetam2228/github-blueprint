# Git vs GitHub

Many beginners think Git and GitHub are the same, but they are different tools that work together.

---

## What is Git

Git is a **version control system** used to track changes in code.

It allows developers to:

- track file changes
- work on different versions of a project
- collaborate without overwriting each other's work

Git works locally on your computer.

Example Git commands:

```
git init
git add
git commit
git push
git pull
```

---

## What is GitHub

GitHub is a **cloud platform that hosts Git repositories**.

It provides additional tools for:

- collaboration
- code review
- issue tracking
- project management
- automation

GitHub allows developers to store their Git repositories online.

---

## Key Differences

| Feature | Git | GitHub |
|------|------|------|
| Type | Version Control System | Cloud Platform |
| Location | Local machine | Online |
| Purpose | Track code changes | Host and collaborate on code |
| Internet Required | No | Yes |

---

## Simple Explanation

Git is the **tool that tracks changes in your code**.

GitHub is the **platform where you store and share that code online**.
