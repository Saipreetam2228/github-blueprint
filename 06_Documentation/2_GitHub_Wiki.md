# GitHub Wiki

GitHub Wiki is a built-in documentation feature that allows developers to create detailed documentation for their repositories.

It works like a simple website where each page can contain guides, tutorials, and explanations about the project.

---

## Why GitHub Wiki is Useful

GitHub Wiki helps developers:

- create detailed documentation
- organize information into multiple pages
- provide guides and tutorials
- explain complex systems

---

## Where Wiki is Used

Wiki is commonly used for:

- project documentation
- API documentation
- installation guides
- developer guides

---

## How to Access the Wiki

To access the Wiki of a repository:

1. Open the repository on GitHub
2. Click the **Wiki** tab
3. Create a new page

---

## Example Wiki Pages

A project wiki may include pages such as:

```
Home
Installation Guide
API Documentation
Developer Guide
FAQ
```

Each page can be linked together to form a complete documentation system.

---

## Difference Between README and Wiki

| Feature | README | Wiki |
|------|------|------|
| Purpose | Quick project overview | Detailed documentation |
| Pages | Single file | Multiple pages |
| Complexity | Simple | More detailed |

---

## Benefits

Using GitHub Wiki provides:

- structured documentation
- easier navigation for users
- better project understanding
