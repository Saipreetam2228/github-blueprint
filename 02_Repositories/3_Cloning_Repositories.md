# Cloning Repositories

Cloning a repository means creating a copy of a GitHub repository on your local computer.

This allows developers to work on the project locally.

---

## Why Cloning is Important

Cloning allows developers to:

- download project files
- work on code locally
- track changes using Git
- contribute to projects

---

## How to Clone a Repository

### Step 1

Open the repository on GitHub.

### Step 2

Click the **Code** button.

### Step 3

Copy the repository URL.

Example:

```
https://github.com/username/repository-name.git
```

### Step 4

Open your terminal or command prompt.

### Step 5

Run the command:

```
git clone https://github.com/username/repository-name.git
```

---

## What Happens After Cloning

After cloning:

- The entire project is downloaded.
- The commit history is preserved.
- Git is connected to the remote repository.

---

## Example

```
git clone https://github.com/user/github-blueprint.git
```

This creates a local copy of the repository.
