# Creating Repositories on GitHub

Creating a repository is the first step in starting a project on GitHub.

A repository stores your project files, documentation, and version history.

---

## Steps to Create a Repository

1. Sign in to your GitHub account.
2. Click the **"+" icon** in the top-right corner.
3. Select **New repository**.
4. Enter a repository name.
5. Add a description (optional but recommended).
6. Choose repository visibility.

- Public
- Private

7. Optionally add:

- README file
- .gitignore
- License

8. Click **Create repository**.

---

## Important Settings

### Repository Name
The name should clearly describe the project.

Example:
```
github-blueprint
```

---

### Description
A short summary explaining what the repository is about.

Example:
```
A complete blueprint for mastering GitHub.
```

---

### Public vs Private

Public repositories are visible to everyone.

Private repositories are only accessible to authorized collaborators.

---

## Best Practices

When creating a repository:

- Choose a clear name
- Add a helpful description
- Include a README file
- Select an appropriate license
